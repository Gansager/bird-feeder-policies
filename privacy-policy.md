# Privacy Policy for bird-feeder

Last updated: March 14, 2026

## Overview
bird-feeder is a personal automation tool that posts bird feeder nature videos to TikTok.

## Data Collection
This application does not collect, store, or share any personal data from users or viewers.

## TikTok API Usage
This app uses the TikTok Content Posting API solely to upload video content 
to the authorized TikTok account. No user data is stored or transmitted 
to third parties beyond what is required by the TikTok API.

## Authentication
OAuth 2.0 tokens used for TikTok authentication are stored locally and 
are never shared with third parties.
